# Tactical Tanks

### A Unity-Based Multiplayer AR Tank Battle Game

## Overview

**Tactical Tanks** is a multiplayer **augmented reality (AR)** tank battle game developed using **Unity** and **Unity Netcode**. The project focuses on real-time multiplayer interaction, combat synchronization, and location-based gameplay within an AR environment.

The game allows players to battle using tanks in the real world, with gameplay restricted to **specific geotagged locations**, ensuring fair and immersive AR combat.
